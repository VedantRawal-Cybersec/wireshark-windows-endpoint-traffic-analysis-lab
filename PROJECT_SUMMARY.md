# Project Summary

## Project Name

Windows Wireshark Endpoint Traffic Analysis Lab

## One-Line Summary

A Windows-based Wireshark lab focused on endpoint traffic capture, DNS analysis, encrypted traffic metadata, local discovery protocols, and privacy-safe reporting.

## What Was Analyzed

- DNS query behavior
- Domain lookup patterns
- Encrypted traffic over port 443
- SSDP local discovery traffic
- Local name resolution/discovery protocols
- Endpoint-to-router/local-device communication

## What Was Protected

The public repository avoids exposing raw sensitive data such as:

- Full device identifiers
- MAC address details
- Raw packet bytes
- Unnecessary local network identifiers
- Other users' traffic

## Skills Demonstrated

- Wireshark display filtering
- Protocol identification
- DNS traffic analysis
- Encrypted traffic metadata interpretation
- Local discovery protocol recognition
- Ethical analysis scope definition
- Structured reporting

## Positioning

This project demonstrates foundational network traffic analysis and documentation skill suitable for SOC, cybersecurity analyst, and blue-team learning paths.
